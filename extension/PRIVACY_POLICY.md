# DualMind Browser Extension - Privacy Policy

**Last Updated**: November 3, 2024

## Overview

DualMind ("we", "our", or "the extension") is committed to protecting your privacy. This privacy policy explains how DualMind handles user data.

## Our Privacy Commitment

**DualMind does NOT collect, transmit, store, or sell any personal data to third parties.** All data remains on your local device.

## Data Storage and Usage

### What We Store Locally (in your browser only)

The extension stores the following data **locally in your browser** using Chrome's storage API:

1. **Chat History**
   - Your conversations with AI
   - Stored locally, never transmitted to us
   - Can be deleted at any time through the extension

2. **User Preferences**
   - Theme selection (dark/light mode)
   - Chosen mode (Local or Cloud)
   - Interface preferences
   - Can be reset at any time

3. **API Keys** (Cloud Mode only)
   - Stored **encrypted** using Chrome's secure storage
   - Never transmitted to us
   - Only sent directly to your chosen AI provider
   - Can be removed at any time

4. **Settings**
   - Extension configuration
   - Default models
   - Custom shortcuts

When using Local Mode:
- AI models run **entirely in your browser**
- No data is sent to any external server
- Complete privacy - your conversations never leave your device
- No internet connection required after initial model download

## Permissions Explained

DualMind requests the following permissions:

### storage
**Why**: To save your settings, chat history, and API keys locally in your browser  
**What it accesses**: Chrome's local storage API (data stays on your device)

### activeTab
**Why**: To read selected text when you use "Ask DualMind" from the context menu  
**What it accesses**: Only the text you've selected, only when you explicitly use the feature

### contextMenus
**Why**: To add "Ask DualMind" options to your right-click menu  
**What it accesses**: Ability to create menu items - no data access

### host_permissions (https://*/*)
**Why**: To enable context menu on all websites  
**What it accesses**: Allows the extension to work on any page you visit, but only when you activate it

## Data Security

- No data transmission to DualMind servers (we don't have any!)
